### Dbug

#### Regular Logs
```csharp
Dbug.Log(msg);
Dbug.Warning(msg);
Dbug.Error(msg);
```
#### Colored Logs
```csharp
Dbug.Red(msg);
Dbug.Orange(msg);
Dbug.Yellow(msg);
Dbug.Green(msg);
Dbug.Blue(msg);
Dbug.Indigo(msg);
Dbug.Violet(msg);
```
#### Stylized Logs
```csharp
Dbug.Bold(msg);
Dbug.Italic(msg);
Dbug.Underline(msg);
Dbug.Strikethrough(msg);
```
```csharp
Dbug.Emphasis(msg);
```

#### Chose your flavor.
```csharp
Dbug.MyLog(msg, #FFFFFF, false, false, false, false);
```
`msg, #hexcolor, bold, italic, underline, strikethrough`
